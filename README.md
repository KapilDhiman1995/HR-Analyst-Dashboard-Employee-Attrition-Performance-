# HR-Analyst-Dashboard-Employee-Attrition-Performance-
This project explores the  HR Analytics Employee Attrition & Performance dataset to uncover the factors that influence employee attrition and overall workforce performance. Through exploratory data analysis (EDA), data visualization, feature engineering, and predictive modeling, the project provides actionable insights that can help organizations improve employee retention and make data-driven HR decisions.
Exploratory Data Analysis (EDA)
Interactive visualizations and insights
Data preprocessing and feature engineering
Machine Learning models for attrition prediction
Performance evaluation using classification metrics
Business recommendations based on analytical findings
